# checkmoney

> Simple and powerful money tracker.

### Installation

```sh
$ yarn
$ cd back yarn evolutions:init
$ cd back yarn evolutions:run
$ cd back yarn start:back:dev
$ cd front yarn start
```

### Usage

```sh
$ cd back yarn start:back:dev
$ cd front yarn start
```

### Commit

```sh
$ yarn s cz
```

**app runnig on localhost:3001**
